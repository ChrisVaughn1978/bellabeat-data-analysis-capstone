Bellabeat Activity Analysis Case Study

Objective:
Analyze user activity data to identify behavior patterns and provide business recommendations.

Tools used:
- Excel
- PowerPoint
- Data analysis and visualization

Key findings:
- Users average 6,547 daily steps vs 10,000 recommended
- Activity levels significantly below recommended targets
- Strong inverse relationship between activity and sedentary time

Recommendations:
Implement engagement features such as activity reminders, milestone rewards, and personalized goals.

Outcome:
Demonstrates ability to analyze behavioral data and communicate executive-level insights.
