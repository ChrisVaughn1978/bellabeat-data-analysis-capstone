# Bellabeat Data Analysis Capstone

## Objective
Analyze Bellabeat user activity data to identify trends and provide business recommendations.

## Tools Used
- Excel
- PowerPoint
- GitHub

## Deliverables
- Executive presentation (Bellabeat Activity &.pptx)
- Dataset (dailyActivity_merged.xlsx)
- Business recommendations

## Key Findings
- Users average 6,547 daily steps vs recommended 10,000
- Users average 17 very active minutes vs recommended 21
- Higher activity correlates with lower sedentary time

## Business Recommendations
- Step-based engagement systems
- Inactivity alerts
- Milestone rewards
- Personalized activity goals
